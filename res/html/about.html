APK Studio is a open-source and cross-platform IDE for reverse-engineering Android applications. Written using Qt framework, it is collaboratively developed by Vaibhav Pandey along with other contributor(s) and currently maintained and developed by Surendra Kumar and contributors.

The main aim of this project is to provide ROM makers with the ease of GUI for customization of pre-packaged applications and Android system. The source code for this application is hosted at Github.
